QUADERN
Com funciona?
Aquest control permet definir metainformació del quadern, com ara, la materia, el nivell o l'autor.
Per editar cadascuna d'aquestes entrades d'informació (que anomenarem etiquetes), serà necessari seleccionar-la i clicar sobre el botó 'editar'.
Cada item d'informació addicional té un nom (o etiqueta) i un valor, tal i com es mostra a la següent imatge:
[IMATGE]

Per defecte, quan es crea un quadern amb l'editor apareixen les següents etiquetes, que s'hauran d'omplir amb la informació correcta del quadern:
[ETIQUETES]
