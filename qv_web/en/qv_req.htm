REQUISITOS TÉCNICOS

El software necesario para acceder a las aplicaciones de Quaderns Virtuals son:

  • Connexión a internet
    Las aplicaciones sólo están disponibles mediante Internet, es decir, no hay ninguna de ellas que disponga de una versión local que se pueda instalar en el ordenador y permita trabajar sin conexión.
  • Navegador web
    El acceso al editor y al visualizador se hace mediante un navegador web como Mozilla FireFox, Internet Explorer (versión 5.5 y superior), Safari, Opera...
  • Motor de Java.
    Algunas preguntas de un cuaderno pueden necesitar cargar applets de Java™. Para ejecutar estos applets es necesario que el ordenador disponga de un programario especial, llamado máquina virtual Java™. La máquina virtual Java™ necesaria para los Quaderns Virtuals tiene que ser de la versión 1.1 o superior.
    En esta página del JClic encontraréis información para instalar y configurar el Java™.
  • Reproductor multimedia.
    Para poder escuchar y visualizar correctamente los audios y vídeos que pueda contener cada cuaderno será necesario tener instalado un reproductor multimedia (como el Real Player o el Quicktime). Además, si el cuaderno tiene incrustada alguna animación flash, será necesario instalar también el plugin de Flash.
