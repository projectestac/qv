Quaderns Virtuals

Quaderns virtuals es un entorno de aprendizaje digital que permite crear diferentes tipos de preguntas para evaluar el progreso y rendimiento de los estudiantes.

Cada cuaderno está formado por una o más hojas y, al mismo tiempo, cada hoja puede contener una o más preguntas sobre una temática concreta. Además, las hojas pueden tener puntos de diálogo, que permiten al alumnado y profesorado escribir comentarios y dudas.

Hay diferentes tipos de preguntas que puede contener un cuaderno, como, preguntas tipo test, preguntas de rellenar espacios, selecciones sobre imágenes, ordenación, etc.

Quaderns Virtuals se basa en diferentes estándares, como QTI (versión 1.2) y Celebrate, para favorecer el intercambio y compatibilidad entre diferentes sistemas.

El proyecto consta de dos aplicaciones: un editor i un visualizador. Con el editor, el profesorado diseña y edita de forma sencilla sus propios cuadernos. En cambio, el visualizador permite a los educandos consultar el contenido propuesto por el profesorado, responder a las preguntas establecidas y ver, si así lo establece el profesor, la corrección automática de los cuadernos.
