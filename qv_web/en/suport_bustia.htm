Contact

We invite you to send any detected error, doubt or suggestion to the Quaderns Virtuals e-mail. This way you will collaborate in the improvement of the tool.

When you send a message make sure that you have correctly configured the sender address in the mail program, otherwise, it would be impossible for you to receive the answers to your messages. It is also recommendable to write your e-mail at the end of the message text, specially if different users use the same computer.

If you ask about the operation of the applications or the activities, make sure you provide all the information you have about the navigator, type of computer and operating system, the error messages that may appear. This will be very useful for us to give you assistance.

 

You can send any doubt, comment or suggestion to our e-mail: qv@xtec.cat
