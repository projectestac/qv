Editor of Quaderns Virtuals

The editor of Quaderns Virtuals allows to teachers to create, edit and publish their own workbooks.

Each workbook is respresented by an XML file and set of multimedia elements (images, videos, sounds). This XML file, that follows standard QTI, defines the page structure and questions of workbook as well as their evaluation. Therefore, the editor allows to create and edit workbooks without knowing XML.

The access to the editor is gained through Internet and it is necessary to be registered as user in edu365.com. Once in the publisher, every teacher has 5MB of space in the server, where he will store the workbooks he creates and the multimedia contents (images, animations, videos, sounds...) that every workbook needs.

The editor is developing to improve and to extend its functions, but it has a current stable version which allows creating and editing in a visual and intuitive way all kinds of activities of Quaderns Virtuals. Anyway, if you have any comment, suggestion or proposal of improvement you can send it to our contact.
