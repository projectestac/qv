Quaderns Virtuals (QV) es un entorno de aprendizaje digital que nace y se desarrolla en el Departament d'Educació de la Generalitat de Catalunya con la voluntad de llevar a la red los cuadernos de ejercicios tradicionales. El proyecto está formado por un conjunto de aplicaciones de software libre que facilitan al profesorado de cualquier nivel y área educativos la creación de recursos didácticos multimedia para evaluar el progreso y rendimiento del alumnado.

Las principales secciones del web son:

  • Quaderns Virtuals
    El proyecto consta de diferentes aplicaciones web (editor, visualizador, generador web...) desarrolladas en la plataforma Java que requieren conexión a Internet y que funcionan sobre diferentes sistemas operativos (Windows, Linux, Mac OS y Solaris).
  • Biblioteca de actividades
    Recurso que está en desarrollo y que pretende alojar los cuadernos creados por educadores y educadoras que quieran compartir su trabajo. Desde esta sección podéis consultar la base de datos de cuadernos e informaros sobre cómo colaborar en el proyecto.
  • Documentos
    Todo tipo de documentación sobre Quaderns Virtuals: artículos, guías de usuario, cursos de creación de actividades...
  • Soporte
    En este apartado encontraréis las preguntas más frecuentes sobre el proyecto Quaderns Virtuals y sus aplicaciones, y el acceso al buzón para hacernos llegar las dudas que no encontréis resueltas en las preguntas frecuentes.
  • Comunidad
    Un espacio para empezar a constituir una comunidad de personas e instituciones interesadas en el proyecto. Contiene todas las novedades y noticias vinculadas a Quaderns Virtuals así como una relación de enlaces a otros webs relacionadas con el proyecto.
