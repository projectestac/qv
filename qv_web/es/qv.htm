Quaderns Virtuals

Quaderns virtuals es un entorno de aprendizaje digital que permite crear diferentes tipos de actividades para evaluar el progreso y rendimiento de los estudiantes.

El proyecto consta de diferentes aplicaciones ( editor, visualizador, generador web...), accesibles desde cualquier navegador web con conexión a Internet.
