Editor de Quaderns Virtuals

El Editor de Quaderns Virtuals es la herramienta que facilita al profesorado la creación, edición y publicación de cuadernos.

Cada cuaderno está representado por un fichero XML y un conjunto de elementos multimedia (imágenes, vídeos, audios...). Este fichero XML, que sigue el estandar QTI, define la estructura de las hojas y preguntas del cuaderno así como su evaluación. El editor permite, por tanto, la creación y edición de un cuaderno, sin necesidad de tener ningún conocimiento de XML.

El acceso al editor se hace mediante Internet (http://clic.xtec.net/qv_editor) y es necesario autenticarse como usuario del edu365.com (servicio de Internet que el Departament d'Educació pone a disposición de toda la comunidad educativa catalana). Una vez dentro del editor, cada profesor dispone de 5MB de espacio en el servidor, donde irá almacenando los cuadernos que vaya creando y los contenidos multimedia (imágenes, animaciones, vídeos, audios...) que necesite cada cuaderno.

El editor está en desarrollo para mejora y ampliar sus funcionalidades, pero dispone actualmente de una versión estable que permite la creación y edición de forma visual e intuitiva de todos los tipos de actividades de Quaderns Virtuals. De todas formas, si tenéis cualquier comentario, sugerencia o propuesta de mejora lo podéis enviar a nuestro buzón (qv@xtec.net).
